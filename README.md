### Deploy

1. Clone project
```
git clone
```
2. Install packages
```
npm install
```
3. Compile project
```
npx hardhat compile
```
4. Uncomment the line in hardhat.config.js
```
   // require('./tasks/tasks');
```
6. Create file .env
```
echo "CONTRACT_ADDR=0x07d8b075b94F76263f8ED38c79FbCb704e648484" > .env
```
7. **npx hardhat**
